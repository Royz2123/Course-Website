מה נלמד?

שלום, איך קוראים לך? מהו מקום מגוריך?
שאלות אלה הן דוגמה לקלט מהמשתמש, כלומר מידע שנרצה לקלוט מהמשתמש לתוכנית שאנו כותבים.
הפקודה ()input מאפשרת לקבל קלט מהמשתמש. בהרצת פקודה זו, התוכנית תחכה עד שהמשתמש ילחץ Enter. לרוב נגדיר משתנה, שיצביע על האובייקט שקלטנו מהמשתמש, כדי שנוכל לעשות בו שימוש חוזר. שימו לב שפייתון מתייחסת לקלט כאל מחרוזת.

דוגמה לשימוש בפקודה input, הגדרת משתנה עבור הקלט, והדפסת המחרוזת שהתקבלה:

>>> name = input("Please enter your name:")
Please enter your name: Pinocchio!
>>> print("Hello", name)
Hello Pinocchio!

המשתנה האוטומטי להמשך קריאה

בעת כתיבת קוד באינטרפרטר, תוכלו להיעזר במשתנה שנקרא "_" (קו תחתי). במשתנה זה נשמרת באופן אוטומטי התוצאה של הפעולה האחרונה שהתבצעה.

דוגמה לשימוש במשתנה _ :

>>> 6
>>> _
6
>>> _ * 2
12

בואו ננסה לקלוט מהמשתמש מספר ולהדפיס את ריבועו.

מה דעתכם על התוכנית הבאה? מה לדעתכם יודפס על המסך?
>>> number = input('Please enter a number:')
Please enter a number: 5
>>> print(number ** 2)

התשובה נכונה, שימו לב לפלט שמתקבל.

מתקבלת הודעת שגיאה מסוג TypeError. הודעת השגיאה אומרת לנו שפעולת העלאה בחזקה על מחרוזת איננה נתמכת. להזכירכם, הפעולה input החזירה לנו ערך מטיפוס str.

כיצד נוכל לפתור את הבעיה?
נוכל להסב את המשתנה מטיפוס str לטיפוס int באופן הבא:

>>> print(int(number) ** 2)
25



לעיתים נצטרך לשנות את הטיפוס של משתנה שהגדרנו או שקיבלנו. לדוגמה, קלט מן המשתמש נשמר כמחרוזת, אך יתכן שנרצה לבצע עליו חישובים מספריים. או להיפך, מספר שנרצה להדפיס כמחרוזת. השינוי של טיפוס המשתנה נקרא הסבה (casting).

להלן דוגמאות להסבה של משתנים:

>>> num = 4
>>> num = float(num)
>>> num
4.0
>>> num = str(num)
>>> num
'4.0'

נניח שהגדרנו שני משתנים ואנחנו רוצים להחליף בין הערכים שלהם.
נדמיין את פיטר-פן שגר בארץ-לעולם-לא ואת שלגיה שגרה עם הגמדים ביער:

>>> where_peter_pan = 'neverland'
>>> where_snow_white = 'forest'

מה יקרה אם יום אחד יחליפו פיטר-פן ושלגיה ביניהם את מקום המגורים? בפייתון, התהליך פשוט ואינטואיטיבי.
דוגמה להחלפת מקום המגורים:

>>> where_peter_pan, where_snow_white = where_snow_white, where_peter_pan

ובאופן כללי:

>>> var1 = 1
>>> var2 = 2
>>> var1, var2 = var2, var1

החלפה בין משתנים להמשך קריאה

החלפה בין משתנים בשפת פייתון היא פעולה פשוטה ביותר. בואו נשווה את פעולת ההחלפה לתהליך דומה בשפות תכנות אחרות. בשפות כדוגמת C++, C וג'אווה צריך להגדיר משתנה שלישי, זמני, ולהיעזר בו. בהתאמה, קטע הקוד הנדרש לכך אף ארוך יותר.

דוגמה להחלפה בין משתנים בשפת C:

int var1 = 1;
int var2 = 2;
int temp;

temp = var1;
var1 = var2;
var2 = temp;
