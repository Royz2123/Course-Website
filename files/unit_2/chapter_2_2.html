מה נלמד?

טיפוס חשוב ושימושי מאוד בפייתון הוא טיפוס בוליאני (באנגלית boolean, ובקיצור bool). משתנה מטיפוס בוליאני הוא משתנה שמקבל את הערכים אמת או שקר (True/False) בלבד.

דוגמאות למשתנים בוליאניים:

the_sun_will_rise_tomorrow = True
i_am_hungry = False

משתנים בוליאניים משמשים הרבה לשמירת תוצאות של ביטויים לוגיים. ביטוי לוגי הוא ביטוי שהתוצאה שלו היא חד משמעית - אמת או שקר.
דוגמה להשמת תוצאה של ביטוי לוגי למשתנה בוליאני (להזכירכם, בפרק הקודם הגדרנו: 4 = brothers_num):

is_big_family = brothers_num > 20

בדוגמה זו המשתנה is_big_family מקבל ערך בהתאם לאמיתות הביטוי הלוגי "מספר האחים גדול מ-20". במקרה שלנו מספר האחים אינו גדול מ-20 ולכן המשתנה יצביע על הערך False.

>>> is_big_family
False

שימו לב, פייתון תתייחס לכל מספר שאינו המספר 0 (אפס) בתור הערך הבוליאני True, ואילו המספר 0 תמיד מייצג את הערך הבוליאני False.

לצורך כתיבת ביטויים לוגיים משתמשים באופרטורים לוגיים.
לפניכם טבלה עם דוגמאות לאופרטורים לוגיים ולאופן השימוש בהם. בדוגמאות הניחו שהגדרנו את המשתנים:

>>> a = 2
>>> b = 5
אופרטור תיאור דוגמה לביטוי לוגי תוצאת הביטוי לדוגמה
== שווה a == 2 True
!= לא שווה b != 5 False
> גדול מ- a > 4 False
< קטן מ- a < 4 True
and True אם תוצאת שני הביטויים היא True a > 3 and b > 4 False
or True אם תוצאת אחד מהביטויים (או שניהם) היא True a > 3 or b > 4 True
not הופך מ-True ל-False, ולהיפך not(a > 4) True
in בדיקת הכלה במחרוזות (ובהמשך בטיפוסים נוספים)
"duck" in "duckling"
"s" in "duckling"
True
False

אפשר גם לשלב ביטויים לוגיים פשוטים ליצירת ביטויים לוגיים מורכבים.
דוגמאות לביטויים לוגיים מורכבים (להזכירכם, הגדרנו 2 = a):

דוגמה לביטוי לוגי תוצאת הביטוי לדוגמה
a == 2 or a < 7 True
(a == 2) and (a < 7) and (a > 0) True
0 <= a <= 1 False
1 < a < 7 < 1 False

ביטויים לוגיים להמשך קריאה

הביטו על הטבלה עם הדוגמאות לעיל ושימו לב לשני דגשים בצורת הכתיבה.
דגשים אלה נועדו בעיקר להקל על הקריאה:
  א. נכתוב רווחים משני צידי האופרטורים.
  ב. בביטויים מורכבים נתחום את חלקיו של הביטוי הלוגי בסוגריים.



תיעוד והפקודה help להמשך קריאה

בהמשך הקורס ניתקל בפקודות רבות שאיננו יודעים מה הן מבצעות ואין עליהן פירוט בקורס. נוכל לבקש תיעוד (כלומר הסבר, פירוט) על פקודה מסוימת בעזרת הפקודה help.

נסו לכתוב באינטרפרטר help ותיכנסו לממשק עזרה אינטראקטיבי:

>>> help

תוכלו גם לבקש עזרה עבור פקודה, ספרייה או מילות מפתח בפייתון אם תוסיפו אותו לפקודת help בסוגריים ומוקף במרכאות.

דוגמה לקבלת תיעוד על האופרטור הלוגי in:

>>> help("in")

נקבל את הפירוט הבא:

The operators "in" and "not in" test for membership.
"x in s" evaluates to true if *x* is a member of *s*, and false otherwise.

בהמשך התיעוד, נמצא גם הסבר רלוונטי לטיפוס מחרוזת:

For the string and bytes types, "x in y" is true if and only if *x* is a substring of *y*

* טיפ לקריאת תעוד בעזרת הפקודה ()help: תוכלו לדפדף קדימה בעזרת מקש רווח, או לצאת באמצע מן התיעוד בלחיצה על q.

בנוסף, קיים תיעוד רב באינטרנט אשר יכול לסייע בהבנת פקודות שאנו לא מכירים. התיעוד כתוב לרוב בפורמט סטנדרטי שתלמדו להכיר עם הזמן. תוכלו למצוא תיעוד גם עבור פעולות בסיסיות של שפת פייתון וגם עבור חבילות הרחבה בהן תיתקלו בהמשך. דוגמה לתיעוד מפורט תוכלו למצוא באתר הבא: https://docs.python.org/3/contents.html.

מומלץ להיעזר באינטרנט על מנת למצוא תיעוד נוסף, אך חשוב לשים לב שהוא כתוב עבור גרסת הפייתון הרלוונטית.
