מה נלמד?

ביחידה הקודמת נתקלנו במשתנה מטיפוס מחרוזת. ביחידה זו נכיר את המחרוזות יותר לעומק.
להזכירכם, מחרוזת הנה טיפוס נתונים מובנה בשפה שנקרא string, או בקיצור str. בפייתון כל רצף של תו אחד או יותר הנו מחרוזת, וכדי להגדירו עלינו להקיף את הטקסט שאנו רושמים במרכאות כפולות (" ") או במרכאות יחידות (' ').
המרכאות מסמנות לפייתון שמדובר באובייקט מטיפוס מחרוזת ולא בשם של פקודה או משתנה כלשהו.


SyntaxError להמשך קריאה

מתכנת בתחילת דרכו כתב תוכנית פשוטה עם המחרוזת הבאה, מה לדעתכם יקרה בעת הגדרתה?

>>> wish = 'I really hope defining this string will work!"
האם המשאלה תתגשם?
לצערנו לא, בעת ההרצה מתקבלת הודעת שגיאה.
אכן, בעת ההרצה מתקבלת הודעת שגיאה. הבה נביט ב-Traceback שמתקבל וננתח אותו.
>>> wish = 'I really hope defining this string will work!"
File "<stdin>", line 1
wish = 'I really hope defining this string will work!"
                                                     ^
SyntaxError: EOL while scanning string literal

הפלט מרמז לנו שמקור הבעיה בקוד הוא בשורה מספר 1, והסמן ^ מראה לנו את מיקום הבעיה בשורה זו.

SyntaxError זוהי שגיאה תחבירית, מהנפוצות בפייתון, וסביר להניח שתיתקלו בה לא מעט במהלך לימוד השפה. שגיאת תחביר מתייחסת לאופן שבו הקוד מסודר, בדגש על מבנה השפה וחוקיה: הזחות, סוגריים מתאימים ועוד. במצב של שגיאה כזו, בלתי אפשרי לתרגם את הקוד לפקודות פייתון חוקיות.

השגיאה שקיבלנו היא: EOL while scanning string literal.

EOL הנו קיצור ל-"end of line". שגיאת EOL משמעותה שפייתון הגיעה לסוף השורה במהלך סריקת המחרוזת. שגיאה זו עלולה להיגרם מסיבות שונות ביניהן: מרכאות או סוגריים שנשכחו, או ניסיון לכתוב מחרוזת ביותר משורה אחת.
במקרה שלנו השגיאה נגרמה בשל חוסר תאימות: המחרוזת הוגדרה בתחילתה עם מרכאות יחידות ולאחר סריקת המחרוזת פייתון הגיעה לסוף השורה וטרם מצאה מרכאות יחידות שסוגרות את המחרוזת. נוכל לתקן ולהחליף את המרכאות בתחילת המחרוזת או בסופה, על מנת לשמור על תאימות.




אל תשכחו, אפשר לבחור את אחת משתי הצורות להגדרת מחרוזת כל עוד אנחנו עקביים עבור מחרוזת מסוימת.
דוגמאות להגדרת מחרוזת בעזרת מרכאות (שתי הצורות אפשריות):

>>> message = "I am a good string"
>>> message
'I am a good string'
>>> message_two = 'I am a good string too!'
>>> message_two
'I am a good string too!'

SyntaxError: invalid syntax להמשך קריאה

מה לדעתכם יקרה בעת הרצת קטע הקוד הבא?

>>> i_am_confused = 'I can't figure out what is wrong here'
File "<stdin>", line 1
      i_am_confused = 'I can't figure out what is wrong here'
                             ^
SyntaxError: invalid syntax

בהרצת התוכנית התקבלה הודעת שגיאה מסוג SyntaxError.
הבה נביט ב-Traceback וננתח אותו:

מיקום הסמן (^) הנו לאחר גרש הכתוב באמצע המחרוזת. נבחין בכך שהטקסט מוקף במרכאות יחידות משני צידיו ובנוסף אליהן קיים גרש בתוכו. למעשה המחרוזת שפייתון קוראת היא 'I can', ואז מסתיימת המחרוזת באופן רשמי ופייתון נתקעת באות t שמופיעה בהמשך השורה.
שגיאה מסוג זה היא שגיאת תחביר ומכונה invalid syntax.


שגיאה זו מעלה את השאלה: מה עושים כאשר מעוניינים שהמחרוזת עצמה תכלול גרש או מרכאות?

הפתרונות הבאים אפשריים:

  • אם המחרוזת מכילה מרכאות כפולות, נתחום אותה במרכאות יחידות.
    דוגמה למחרוזת שמכילה מרכאות כפולות:

    >>> '"All in!", she said happily.'
    '"All in!", she said happily.'
  • אם המחרוזת מכילה גרש, נתחום אותה במרכאות כפולות.
    דוגמה למחרוזת שמכילה גרש:

    >>> "Don't do it. You will lose the poker game."
    'Don't do it. You will lose the poker game.'
  • אם המחרוזת מכילה גם מרכאות וגם גרש, נוכל להיעזר בתו מיוחד שמסמן לפייתון להתייחס למרכאות/לגרש בתוך המחרוזת כאל תווי טקסט רגילים.
    התו המיוחד נקרא לוכסן הפוך, או backslash ומכונה גם ה-Escape character. הלוכסן ההפוך נראה כך \ ונכתוב אותו לפני כל מרכאות או גרש שבתוך המחרוזת. שימושים נוספים של הלוכסן ההפוך נפרט בהמשך.
    דוגמאות לשימוש בלוכסן הפוך בהדפסת מחרוזות:

    >>> print("\"Play with me\", she said.\"It will be SO hilarious when you will lose!\"")
    "Play with me", she said." It will be SO hilarious when you will lose!"

    >>> '\'She is so tough!\', he thought to himself.'
    "'She is so tough!', he thought to himself."

אפשר ליצור מחרוזות על ידי שרשור מחרוזות קיימות, באמצעות סימן החיבור (הסימן +).
דוגמה לשרשור מחרוזות:

>>> str1 = "We "
>>> str2 = "belong "
>>> str3 = "together"
>>> print(str1 + str2 + str3)
We belong together

אפשר להכפיל מחרוזת באמצעות סימן הכפל (הסימן *) ובכך ליצור מחרוזת חדשה.
דוגמה להכפלת מחרוזת:

>>> duplicate_me = ":-) "
>>> print(duplicate_me * 8)
:-) :-) :-) :-) :-) :-) :-) :-)

אפשר להסב משתנה שאינו מטיפוס מחרוזת בעזרת הפקודה ()str, כלומר להפוך אותו לטיפוס מחרוזת.
דוגמה להסבת משתנה למחרוזת:

>>> x = 3
>>> type(x)
<class 'int'>
>>> x + x
6
>>> x = str(x)
>>> x
'3'
>>> type(x)
<class 'str'>
>>> x + x
'33'
