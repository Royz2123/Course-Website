מה נלמד?

ביחידות הקודמות למדנו לעבוד עם אובייקטים מטיפוסים פשוטים יחסית כמו float, int ו-string.
ביחידה זו נכיר פתרון למקרים בהם נרצה לאחד מספר אובייקטים כך שנוכל להתייחס אליהם יחד.
הרשימה היא מבנה נתונים (data structure) שימושי מאוד, המתאים לתפקיד זה.
לדוגמה, נוכל ליצור רשימה של שמות סרטים, בה כל איבר הוא אובייקט מטיפוס מחרוזת (כגון 'Titanic', 'Star Wars') או רשימה של הגילים של כל בני המשפחה, בה כל איבר הוא מטיפוס int, ועוד.

רשימה מסמנים על ידי סוגריים מרובעים [ ]. בכתיבת סוגריים מרובעים אפשר ליצור רשימה, אפילו ריקה.
דוגמה ליצירת רשימה ריקה:

>>> empty_list = []
>>> type(empty_list)
<class 'list'>
 >>> print(empty_list)
[]

כדי ליצור רשימה המכילה אובייקטים, יש להפריד אותם באמצעות פסיקים. כל אובייקט מכונה איבר ברשימה.
דוגמה ליצירת רשימה המכילה את האיברים הראשונים בסדרת פיבונאצ'י (סדרה בה כל איבר הוא סכום שני האיברים הקודמים לו):

>>> fibo_list = [1, 1, 2, 3, 5, 8, 13, 21]
>>> fibo_list
[1, 1, 2, 3, 5, 8, 13, 21]

הרשימה fibo_list מכילה שמונה מספרים, כלומר איברים מהטיפוס int, אולם איברים ברשימה יכולים להיות גם מטיפוסים שונים.
דוגמה לרשימה עם איברים מטיפוסים שונים:

>>> crazy_list = [2, "The Big Bang Theory!", True, 4.6]
>>> crazy_list
[2, 'The Big Bang Theory!', True, 4.6]

כל איבר ברשימה מוגדר על ידי מיקום (אינדקס). כלומר, קיימת חשיבות לסדר האיברים ברשימה.
בדומה למחרוזת, גם ברשימה המיקומים מתחילים מאפס, ואפשר לגשת לאיבר מסוים על ידי ציון המיקום שלו.

דוגמה לגישה לאיברים ברשימה:

>>> crazy_list[0]
2
>>> crazy_list[1]
'The Big Bang Theory!'
>>> crazy_list[2]
True
>>> crazy_list[3]
4.6

כפי שראינו, רשימה יכולה להכיל איברים מטיפוסים שונים. בהתאמה לכך, רשימה יכולה אף להכיל איברים מהטיפוס "רשימה".

דוגמה לרשימה המכילה שני איברים שכל אחד מהם רשימה בפני עצמו:

>>> unique = [["a", "list"], ["of", "lists"]]
>>> unique
[['a', 'list'], ['of', 'lists']]

בואו נראה כיצד לגשת לאיברים של רשימה בתוך רשימה.
כאשר יש רשימה "חיצונית" המכילה רשימה "פנימית", ראשית יש לגשת למיקום של האיבר ה"חיצוני" ואז לגשת לאיבר ה"פנימי" הרצוי.

דוגמה לגישה לאיברים ברשימה בתוך רשימה:

>>> unique
[['a', 'list'], ['of', 'lists']]
>>> unique[1]
['of', 'lists']
>>> unique[1][0]
'of'
>>> unique[0][1]
'list'
>>> unique [0][0]
'a'

ייתכן שתהיתם מתי שימושי להשתמש במבנה של רשימה בתוך רשימה. מבנה זה נוח לשימוש כאשר מארגנים יחד נתונים מסוגים שונים.

לדוגמה, לפניכם רשימה בה כל איבר מכיל רשימה שכוללת שם, שנת לידה ותחום עיסוק.
הביטו ברשימה ובאופן הגישה לנתונים ברשימה:

>>> influential_people = [["Mark Zuckerberg", 1984, "Computer programmer"], ["Sheryl Sandberg", 1969, "Technology executive"], ["Albert Einstein", 1879, "Theoretical physicist"]]
>>> influential_people[0]
['Mark Zuckerberg', 1984, 'Computer programmer']
>>> influential_people[0][0]
'Mark Zuckerberg'
>>> influential_people[0][1]
1984
כדי לוודא שלא התבלבלתם, איזו מן הפקודות הבאות תדפיס את השם '‏Sheryl Sandberg‏'?
פקודה זו תדפיס ['Albert Einstein', 1879, 'Theoretical physicist'], נסו שוב.
אכן! הידעתם? בין תפקידיה של Sheryl Sandberg: מנהלת התפעול הראשית של פייסבוק!
פקודה זו תדפיס 1969, נסו שוב.
פקודה זו תדפיס 'Albert Einstein', נסו שוב.

למדנו כיצד ליצור רשימה באמצעות סוגריים מרובעים [ ]. דרך נוספת ליצירת רשימה חדשה היא בעזרת הפונקציה list.
דוגמה ליצירת רשימה ריקה באמצעות ()list:

>>> empty_list = list()
>>> empty_list
[]

באמצעות ()list אפשר גם ליצור רשימה ממחרוזת. זו אינה רשימה ריקה, אלא רשימה שכל איבר בה הוא תו במחרוזת המקורית.
דוגמה ליצירת רשימה ממחרוזת:

>>> letters = "science"
>>> list_of_letters = list(letters)
>>> list_of_letters
['s', 'c', 'i', 'e', 'n', 'c', 'e']
שמירת איברי רשימה למשתנים להמשך קריאה

לפניכם טריק שימושי בעבודה עם רשימות: אפשר לשמור בקלות איברי רשימה לתוך משתנים!
דוגמה להשמת איברי רשימה למשתנים:

>>> cool_trick = [1, 2, 3]
>>> a, b, c = cool_trick
>>> a
1
>>> c
3

שימו לב: עליכם להקפיד לייצר משתנים כמספר האיברים ברשימה אחרת מתקבלת הודעת שגיאה!
