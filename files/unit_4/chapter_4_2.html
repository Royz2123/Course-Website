מה נלמד?

ודאי נתקלתם בחייכם בצורך לקבל החלטות, כלומר לבחור איך להתנהג בהינתן תנאים מסוימים. לדוגמה: אם התחזית גשומה אז אקח מטרייה, אם צפויה שמש אצטייד בשמשייה ובכל מזג אוויר אחר - אצא עם שכמייה.

מבנה כזה מכונה תנאי ובתכנות הוא ממומש באמצעות הפקודה if ("אם"). פקודת תנאי (if) מאפשרת שליטה על סדר ביצוע (וגם אי-ביצוע) פעולות בתוכנית. כלומר, היא משמשת לבקרת זרימה במהלך תוכנית.

פקודת תנאי מאפשרת ביצוע פעולה או פעולות, אם (ורק אם) מתקיים תנאי כלשהו. אפשר גם להגדיר ביצוע פעולה אחרת אם (ורק אם) התנאי לא מתקיים.

נכיר את צורת כתיבת התנאי בפייתון.

התנאי if:

את התנאי נתחיל במילת ההתנייה if, מיד אחריה נכתוב מהו התנאי שבודקים (ביטוי לוגי) ובסוף השורה נכתוב נקודתיים. לאחר מכן יתחיל בלוק מוזח. הבלוק מתאר את מה שיקרה בהינתן קיום/אי קיום התנאי ועליו להכיל לפחות שורת קוד אחת.

דוגמה לבלוק תנאי פשוט:

if 3 < 4:
  print("Three is less than four.")
Three is less than four.

הביטו בדוגמה הבאה, מה יקרה כאשר נריץ אותה?

if 1 == 2:
  print("No way!")

בדוגמה זו התנאי הוא הביטוי הלוגי 2 == 1. שימו לב שתנאי זה לעולם לא יתקיים ולכן השורה בבלוק המוזח לא תודפס אף פעם.

else:

לאחר בלוק של פקודת if, שמתבצע במקרים בהם התנאי מתקיים, אפשר להוסיף בלוק שמתבצע כאשר התנאי אינו מתקיים, כלומר בכל המקרים האחרים. בלוק זה יגיע לאחר פקודת else ואחריה נקודתיים.

דוגמה למבנה if else:

name = input()
if name.endswith("son"):
  print("You might be Scandinavian.")
else:
  print("You're probably not Scandinavian.")

elif:

המילה השמורה elif (קיצור של else if, אחרת אם) משמשת להגדרת תנאי ביניים (אחד או יותר). פייתון בודקת נכונות של תנאי כלשהו ואם התנאי לא מתקיים, כלומר הבדיקה הראשונה החזירה שקר, היא ממשיכה לבדיקה הבאה.

elif יכולה להופיע רק לאחר משפט if. בסוף המקרים השונים שנבדקים עם elif אפשר (אך לא הכרחי) לשלב משפט else. משפט זה יתבצע אם אף אחד מהמצבים הקודמים לא התקיים.

דוגמה למבנה if elif else:

choice = input()
if choice == 1:
  print("You've chosen one.")
elif choice == 2:
  print("You've chosen two.")
elif choice == 3:
  print("You've chosen three.")
else:
  print("You've chosen something else.")

שימו לב, השימוש במילה elif מסייע לנו להימנע מכתיבה מקוננת, כלומר תנאי בתוך תנאי. כתיבה מקוננת נחשבת מסורבלת, כפי שאפשר להתרשם מהשוואת הדוגמה הבאה, בה משתמשים ב-if וב-else בלבד, לדוגמה הקודמת (שכוללת גם elif):

choice = input()
if choice == 1:
   print("You've chosen one.")
else:
  if choice == 2:
     print("You've chosen two.")
  else:
    if choice == 3:
       print("You've chosen three.")
    else:
       print("You've chosen something else.")

בפקודות תנאי אפשר לשלב ביטויים לוגיים, כפי שלמדנו ביחידה על משתנים.
דוגמה לשילוב אופרטורים לוגיים ואופרטורים של השוואה בפקודת תנאי:

if (months_passed == 18) and (location_moon == location_sun):
   print("Eclipse!")
else:
   print("Just a regular day…")

בדוגמה זו מספר החודשים צריך להיות שמונה עשר, ומיקום הירח צריך להיות זהה למיקום השמש כדי שיודפס eclipse (ליקוי חמה). אחרת, יודפס Just a regular day (רק יום רגיל…)


ערך ברירת מחדל למשתנה להמשך קריאה

לפניכם קטע קוד מתוך תוכנית שלמה. נסו להבין מה עלול להיות בעייתי בו.

...
pet_num = int(input(‘How many pets do you have?'))
if pet_num > 2:
  message = "You are an animal lover!"
print(message)
...

שימו לב שהמשתנה message מוגדר (או אינו מוגדר) כתלות בביטוי הלוגי pet_num > 2. אם ברשותך תוכי אחד, אזי לאחר ההרצה:

How many pets do you have? 1

במצב כזה, pet_num = 1, התנאי לא מתקיים וכתוצאה מכך גם המשתנה message לא מוגדר.
כל ניסיון להדפיס את message או להשתמש בו בהמשך התוכנית יגרור שגיאה מסוג NameError שמצביעה על כך שהמשתנה message אינו מוגדר.

באופן כללי, שימוש במשתנה שמוגדר (או שאינו מוגדר) כתלות בתנאי לוגי אינו נחשב תכנות טוב.

בכדי לוודא שמשתנה תמיד מוגדר, כדאי לתת לו ערך ברירת מחדל (דיפולט). הרבה יותר קל וקריא לבדוק האם משתנה הוא בעל ערך ברירת מחדל מאשר לבדוק האם המשתנה בכלל קיים.
במקרה שלנו נוכל לאתחל את המשתנה בכל ערך שנרצה, לדוגמה במחרוזת ריקה:

message = ""
