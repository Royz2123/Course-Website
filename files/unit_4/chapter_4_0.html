מה נלמד?

עד כה, לרוב, כתבנו את הקוד שלנו באינטרפרטר. האינטרפרטר מיועד לכתיבת קוד קצר ובדיקת חלקים קטנים של קוד. אך אם נרצה, ובעיקר בתוכניות גדולות, אפשר לכתוב קוד גם בעורך טקסט. בעורך טקסט נוכל לכתוב קוד, לשמור אותו כקובץ בעל הסיומת py, ולהריץ אותו.
עורך טקסט, כשמו, מאפשר עריכה נוחה יותר של הקוד. בכדי לכתוב קבצי פייתון אפשר לעבוד עם כל עורך טקסט, אפילו כתבן (notepad) בסיסי. עם זאת, קיימים עורכי טקסט אשר מסייעים לנו בכתיבת הקוד על ידי סימון חלקי קוד בצבעים, השלמה אוטומטית, הקפדה על קונבנציות ואף עזרה באיתור בעיות בכתיבת הקוד. עורך טקסט כזה מקל ומייעל את העבודה.
בקורס זה נעבוד עם עורך טקסט שנקרא ++Notepad.

להורדת ++Notepad למחשב שלכם לחצו כאן והמשיכו על פי ההוראות:

  1. גללו לתחתית הדף ולחצו על האפשרות Notpad++ Installer 64-bit x64.
  2. ההורדה תחל באופן אוטומטי. אם אינכם יודעים היכן למצוא את הקובץ, הוא ככל הנראה יוצג כך בתיקיית ההורדות שלכם:
    לאחר שתסתיים הורדת הקובץ, הריצו אותו על ידי לחיצה כפולה עליו.
  3. בחרו שפה ולחצו OK.
  4. המשיכו את ההתקנה על פי הנחיות אשף ההתקנה, בסוף לחצו על "סיים".
  5. סיימנו, אתם יכולים להתחיל לכתוב תוכניות בפייתון בעורך הטקסט ++Notepad!

בואו נכתוב תוכנית ראשונה בעורך הטקסט ++Notepad. פתחו את עורך הטקסט ועקבו אחר ההוראות:

  1. כתבו תוכנית בפייתון.
    דוגמה לתוכנית פשוטה שמדפיסה הודעה למסך:

    print("This is my first Python program in Notepad++")
  2. לאחר כתיבת הקוד, שמרו אותו בצורה של קובץ הרצה של פייתון, כלומר קובץ עם הסיומת py.
    שימו לב שמייד לאחר השמירה השתנו הצבעים של חלקים מהקוד, עובדה שמסייעת להבחין בין מרכיבים שונים בשפה.
    זהו אחד היתרונות שבשימוש בעורך טקסט.
  3. כעת תוכלו להריץ את הקובץ ששמרתם מתוך שורת הפקודה (CMD).
    1. פתחו את שורת הפקודה והגיעו אל התיקייה בה שמרתם את הקובץ. לשם כך היעזרו בפקודה cd המאפשרת לשנות את תיקיית העבודה הנוכחית:
      cd <path>
      לדוגמה, אם שמרנו את הקובץ בתוך c:\python נכתוב:
      cd c:\python
    2. לאחר המיקום בתיקייה הנכונה, כתבו python ולאחר מכן את שם הקובץ כולל הסיומת py. :
      python <filename>
      לדוגמה, אם שמרנו את הקובץ שנקרא program.py נכתוב:
      c:\python>python program.py
    3. לחצו Enter להרצת הקובץ.
      והרי ההודעה שמודפסת על המסך:
      c:\python>python program.py
      This is my first Python program in Notepad++

מעתה נשתמש בהדגמות הן בעורך הטקסט, לרוב עבור קטעי קוד ארוכים יותר, והן באינטרפרטר, עבור קטעי קוד קצרים. המשיכו גם אתם להיעזר בשני הכלים ולהכיר את היתרונות של כל אחד.
